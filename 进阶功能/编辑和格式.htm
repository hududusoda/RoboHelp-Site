编辑和格式

 

本章节介绍如何使用 Adobe RoboHelp 的编辑和格式进阶功能，包括使用图像和多媒体、创建和管理链接、创建和使用变量以便更新、带代码段的单个源文件和使用查找和替换。

使用图像和多媒体

使用 RoboHelp 强大的工作区可以轻松地将图像和多媒体添加到你的主题中。 插入各种格式的图像、计算机、YouTube 和 Vimeo中的视频以及音频剪辑。 有了最新的 RoboHelp 界面，你只需点击一下就可以预览YouTube 视频。 你还可以添加图像缩略图，当在输出中单击时可以缩放。 在你体验创作、发布和查看项目输出的最流畅工作流的同时，使用图像和多媒体发布丰富且具有创造性的内容。

添加图像

  1. 在 Author 工具栏中，选择 Content ，此时将打开 Content 面板；

  2. 双击 Content 面板中的主题以打开该主题。将光标放置在要插入图像的位置

  3. 在屏幕顶部的 Standard 工具栏中，单击 Insert Images 图标以添加图像

  4. 在 Insert Images 对话框中，使用下列选项之一选择图像:

    • 项目文件浏览内容或最近使用的下拉部分中的项目文件

    • 本地文件浏览计算机或指定图像文件的路径

    • Web 链接指定图像的 URL

  5. 单击 Insert ，所选的图像将添加到主题中

编辑图像属性

  1. 在打开的主题中，单击所需的图像

  2. 在屏幕右侧的 Properties 面板> General选项卡中，Type 字段显示 img。 面板中的其他字段显示图像的属性

  3. 使用以下字段修改图像的属性:

    • Source 显示从其中上载图像的文件夹的路径

    • Alternate Text 为图像指定 Alternate Text。该领域中使用的术语提高了图像的可搜索性

    • Image Title 指定图像的标题，当你悬停在图像上时显示该标题

    • Width 指定图像的宽度

    • Maintain aspect ratio icon。RoboHelp 会自动维护图像的纵横比。如果更改图像的一个分量（宽度/高度），另一个分量（高度/宽度）会根据纵横比自动更改。 若要禁用维护纵横比，请单击该图标

    • Height 指定图像的高度

    • Reset to default icon 单击重置为图像的原始高度和宽度 

    • Text Wrapping 单击左图标或右图标选择图像相对于文本的对齐方向，默认值为 none

    • 使用Thumbnail Select将图像用作输出中的缩略图，并指定其宽度和高度 

添加视频

你可以通过以下方式在主题中添加视频:

从你的计算机，YouTube或Vimeo

  1. 在 Author 工具栏中，选择 Content 。此时将打开 Content 面板

  2. 双击 Content 面板中的主题以打开该主题。将光标放置在要插入视频的位置

  3. 在 Standard 工具栏中，从下拉列表中选择 Insert Multimedia 图标 和下列选项之一:

    • Video File 从计算机中选择视频

    • YouTube

    • Vimeo

  4. 在 Insert Multimedia 对话框中，使用下列选项之一选择视频:

    • 项目文件浏览内容或最近使用的下拉部分中的项目文件。或者，按文件名快速搜索视频。RoboHelp在 Contents 面板中搜索指定的文件名

    • 本地文件：浏览你的计算机或指定视频的路径

    • Web 链接：指定视频的 URL

  5. 单击 Insert ，所选视频将添加到主题中

导入和插入Adobe Captivate视频

注意:
要能够在主题内容中插入Adobe Captivate视频，必须首先将其导入到项目中。

  1. 在 Author 工具栏中，选择 Content 。 此时将打开 Content 面板

  2. 在 Content 面板中，执行下列操作之一以在项目中导入视频:

    • 选择所需的文件夹，然后选择 File > Import > Adobe Captivate Demo 

    • 右键单击某个文件夹或单击 ，然后选择Import>Adobe Captivate Demo

  3. 在 Open 对话框中，选择要导入的视频

    AdobeCaptivate将视频输出存储为文件和文件夹的集合。 要将所有关联文件自动导入为压缩文件，请从output文件夹中选择index.html或index.htm。 或者，可以选择整个输出文件夹的.zip或.cpz文件

    单击 Open

    RoboHelp将压缩后的视频按文件名的字母顺序以 .cpz 格式放在文件夹中

  4. 若要在内容中插入导入的视频，请双击 Content 面板中的主题。 在打开的主题中，将光标放在要插入视频的位置

  5. 在 Standard 工具栏中，选择 Insert Multimedia 图标 ，然后从下拉列表中选择“Adobe Captivate Demo”

  6. 在 Standard 工具栏中，选择 Insert Multimedia 图标 ，然后从下拉列表中选择“Adobe Captivate Demo”
     在Adobe Captivate Demo对话框中，从Contents下拉部分中，选择导入的视频。

  7. 在插入所选视频之前，你可以在Adobe Captivate Demo对话框中预览它。单击 Insert ，将插入主题中光标的位置

删除图像或视频

要从主题中删除图像或视频，请执行以下操作:

  1. 在 Author 工具栏中，选择 Content 。此时将打开 Content 面板

  2. 在 Content 面板中，打开包含要删除的图像或视频的主题

  3. 选择所需的图像或视频，然后按 Delete

创建和管理链接

链接便于通过简单的单击从一个内容元素快速导航到另一个内容元素。 在RoboHelp中创作时，你可以添加到文本或图像的链接，并将它们指向项目文件，计算机上的文件，网页或电子邮件。

链接类型

RoboHelp中提供了以下类型的链接，以帮助在输出中构建导航:

  • 超文本链接突出显示文本中的链接，单击时打开目标

  • 图像链接图像中的链接，该链接将你引导到目标，或在单击图像时打开弹出窗口

  • 为链接添加书签，该链接将你带到主题中的特定位置

  • 浏览序列在输出中查看的一系列主题

  • 搜索包含搜索关键字的主题在搜索结果中显示为链接

插入链接

要在文本或图像上创建链接，请执行以下步骤:

  1. 在打开的项目中，选择要在其上创建链接的文本或图像

  2. 在 Standard 工具栏中，单击 Insert Link 图标

  3. 在 Link to 对话框中， Display Text 字段显示要向其插入链接的选定文本。如果选择了图像，此字段将保持空白。可以从以下选项中选择要插入的链接类型:

    • Project Files 选择此选项链接到项目内的主题。 使用搜索栏搜索主题。 你还可以使用最近使用的面板或内容面板来选择你所选择的主题

    • Local Files 选择此选项链接到计算机上的文件。单击 以浏览并选择.htm或.html文件。然后，可以在 Link to 对话框中预览所选文件

    • Webpage 选择此选项链接到网页。 在 Enter or Paste URL 字段中，指定要链接到的网页的URL

    • Email 选择此选项链接到用户可以发送的电子邮件。你可以在 Address 字段中指定希望用户发送到的电子邮件地址。也可以在Subject和Body中分别输入邮件的主题和正文。在输出中，当用户单击此链接时，草稿将在默认邮件客户端中打开。用户可以在发送邮件之前进一步编辑邮件

  4. 单击链接。在 Standard 工具栏中单击 Preview  时，可以预览插入的链接

修复断开的链接

你可以快速生成一个断开链接报告，以查看你的项目中需要修复的链接。为此，请执行以下步骤:

  1. 在打开的项目中，单击 Author 工具栏中的 Report

  2. 在 Report 面板中，双击断开的链接。或者，单击断开链接旁边的 Options ，然后选择 Open 

通过参考断开链接报告中的信息来修复断开的链接。此报告显示已移动，重命名或删除的链接列表。你还可以查看包含断开链接及其链接类型的相应主题。

创建和使用变量以便更新

变量是一个文本短语或单词，它充当可重用的信息片段，在项目中的多个主题中同时使用。这使你的内容便于移动和更新。当你修改变量或其值时，该变量或值都会在整个项目中更新。

创建变量

变量是为其赋值的文本短语。按照以下步骤在打开的项目中创建变量:

  1. 在Author工具栏中，单击Variables。 此时将打开 Variables 面板

  2. 在Variables面板中，单击 。 此时将打开 Define Variables 对话框

  3. 在 Define Variables 对话框中，在 Value 中指定变量的名称及其对应的文本值

  4. 单击 Done 

编辑变量

要在RoboHelp中编辑变量，请执行以下步骤:

  1. 在Author工具栏中，单击Variables。此时将打开 Variables 面板

  2. 在 Variables 面板中，单击所需变量旁边的 Options 

  3. 在 Options 列表中选择 Edit 

  4. 在 Edit Variable 对话框中，可以编辑所选变量的名称和值

  5. 单击 Done 

插入变量

在打开的主题或父页中插入或使用变量:

  1. 打开主题:

    在 Author 工具栏中，选择 Content ，然后从 Content 面板中打开你选择的主题

    打开父页:

    在 Author 工具栏中，选择 Master Pages ，然后从 Master Pages 面板中打开所选的父页

  2. 在Author工具栏中选择变量。 此时将打开 Variables 面板

  3. 从 Variables 面板中选择一个变量。 现在将所选变量拖放到打开的主题或父页中所需的文本位置

    或者，按Ctrl+1。在 Variables 对话框中，你可以通过键入前几个字母来筛选变量，按向上和向下箭头键筛选，按 Enter 插入变量

  4. 单击 Standard 工具栏左侧的 Save All 图标

删除变量

  1. 在Author工具栏中，单击Variables，此时将打开 Variables 面板

  2. 在 Variables 面板中，单击所需变量旁边的 Options

  3. 在 Options 列表中选择 Delete 

  4. 在 Confirm 对话框中单击 Yes 以删除所选变量

生成变量报告

你可以生成变量报告来查看和管理项目中的变量。 使用此报告，你可以找到在主题中使用的变量和在任何主题中未使用的变量。 在删除变量之前，可以找到使用该变量的主题，并删除对该变量的引用。

然后可以下载或打印变量报告。

  1. 在 Author 工具栏中，选择 Report 。 此时将打开 Report 面板

  2. 在 Report 面板中，选择 Variables 

  3. 使用屏幕右侧的 Filters 面板自定义报告

  4. 单击 Standard工具中的Download图标 即可立即下载报告。报告保存为 .csv文件

  5. 若要打印报告，请单击 Standard 工具栏中的 Print 图标 

带代码段的单个源文件

一个片段可以是一段文本，一段代码，一个像图像这样的对象，或者整个主题本身。 你可以单个源文件代码段，并在项目或父页中的多个主题之间重用它。代码段还可以包含变量和条件文本标记。 更改代码段的内容时，会在包含它的所有主题和父页上更新。

创建代码段

在RoboHelp中打开一个项目，按照以下步骤创建一个代码段:

  1. 在 Author 工具栏中，选择 Content ，然后从 Content 面板中打开所需的主题

  2. 在Author工具栏中，选择Snippets。 此时将打开 Snippets 面板

  3. 在Snippets面板中，单击 +

  4. 在 New Snippets 对话框中，分别在 Name 和 Description 中指定代码段的名称和说明

  5. 单击 Create 。代码段已创建，可以从 Snippets 面板访问

插入代码段

在打开的主题或父页中插入或使用代码段:

  1. 打开主题:

    在 Author 工具栏中，选择 Content ，然后从 Content 面板中打开你选择的主题

    打开父页:

    在 Author 工具栏中，选择 Master Pages ，然后从 Master Pages 面板中打开所选的父页

  2. 在 Author 工具栏中，选择 Snippets 。在 Snippets 面板中，选择一个代码段

  3. 现在将所选代码段拖放到打开的主题或父页中的首选位置。或者，按Ctrl+2。 在 Snippets 对话框中，可以通过键入前几个字母来筛选代码段。按向上和向下箭头键筛选，按 Enter 插入代码段

  4. 单击 Standard 工具栏左侧的 Save All 图标  

删除代码段

要从项目中删除代码段，请执行以下步骤:

  1. 在Author工具栏中，选择Snippets。 此时将打开 Snippets 面板

  2. 选择要删除的代码段，然后单击代码段旁边的 Options

  3. 在 Options 列表中，选择 Delete 

  4. 在 Confirm 对话框中，单击 Yes 

生成代码段报告

你可以生成一个代码段报告，以查看项目中代码段的详细信息。你还可以自定义，下载或打印报告。

  1. 在 Author 工具栏中，选择 Report 

  2. 在出现的 Report 面板中，选择 Snippets 

  3. 你可以使用屏幕右侧的 Filters 面板自定义报告

  4. 单击 Standard 工具栏中的 Download 图标可立即下载报告

  5. 单击 Standard 工具栏中的 Print 图标以打印报告

使用查找和替换

查找短语或文本

按照以下步骤快速查找主题中的短语或文本:

  1. 在打开的项目中，从 Content 面板中打开所需的主题

  2. 在 Standard 工具栏的左上角，单击 Find and Replace 工具

  3. 此时出现带有 Find 字段的底部面板。 在此字段中，输入要在打开的主题中查找的短语或文本

  4. 单击 Find 。找到的短语或文本在主题中突出显示

注意:
在底部面板中，单击 以执行以下操作:
Ignore Case 选择此选项可忽略文本中的大小写。
Whole word only 选择此选项查找整词。

快速替换短语或文本

按照以下步骤，可以轻松地用主题中的备用短语或文本替换特定短语或文本:

  1. 在打开的项目中，在 Content 面板中打开所需的主题

  2. 在 Standard 工具栏的左上角，单击 Find and Replace 工具

  3. 此时出现带有 Find 字段的底部面板。 在此字段中，输入要在打开的主题中查找的短语或文本

  4. 单击 Find ，找到的短语或文本在主题中突出显示

  5. 在 Replace with 字段中，输入要替换主题内找到的文本的短语或文本

  6. 单击 Replace 更新突出显示的短语或文本。你可以单击 Replace All 以替换打开主题中的指定短语或文本
