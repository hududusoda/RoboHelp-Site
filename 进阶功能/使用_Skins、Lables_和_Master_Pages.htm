使用 Skins、Lables 和 Master Pages

 

本章节介绍如何使用 Adobe RoboHelp 的 Skins、Lables 和 Master Pages 进阶功能。

使用 Skins

什么是 Skins

RoboHelp 中的 skins 用于在发布帮助系统之前更改和自定义帮助系统的外观。 Skins 可以通过更改输出的颜色、按钮、字体、图标、标志和背景来自定义外观。 无论要发布的是响应式 HTML5 还是 WebHelp，RoboHelp 都会在发布窗口的 Output 面板中提供多个预定义的外观。这些预定义外观可以按原样应用，也可以对其进行编辑以实现首选的输出布局。

添加 Skins

  1. 点击标准工具栏左侧的 Output 选项卡

  2. 选择左侧的 skins

  3. 选择 skins 右上角标记 

  4. 在选项框中选择对应选项

编辑 Skins

  1. 点击标准工具栏左侧的 Output 选项卡

  2. 选择左侧的 skins

  3. 选中需要编辑的 skin，点击  并选择 Edit

重命名 Skins

  1. 点击标准工具栏左侧的 Output 选项卡

  2. 选择左侧的 skins

  3. 选中需要编辑的 skin，点击   并选择 Rename 或者按 F2

升级 Skins

升级 RoboHelp Classic 项目时，RoboHelp 会自动升级所有响应式 HTML5 外观。 在 RoboHelp Classic 编辑器之外自定义的外观，请在 RoboHelp 2019 的布局面板中使用组件添加自定义 JavaScript 或 CSS 文件。进入 skin 的编辑界面，选择 Layout 中的 User Assets 即可添加自定义 JavaScript 或 CSS 文件。

  1. 点击标准工具栏左侧的 Output 选项卡

  2. 选择左侧的 skins

  3. 选中需要编辑的 skin，点击  并选择 Edit

  4. 在Layout中选择 User Assets， 即可添加自定义 JavaScript 或 CSS 文件

导入 Skins

  1. 点击标准工具栏左侧的 Output 选项卡

  2. 选择左侧的 skins

  3. 选择 skins 右上角标记 

  4. 在选项框中选择 Import 即可导入 skin 文件（.skz 或 .slz 格式）

导出 Skins

  1. 点击标准工具栏左侧的 Output 选项卡

  2. 选择左侧的 skins

  3. 选中需要导出的 skin，点击  并选择 Export 即可导出 skin 文件到指定的文件夹

删除 Skins

  1. 点击标准工具栏左侧的 Output 选项卡

  2. 选择左侧的 skins

  3. 选中需要删除的 skin，点击  并选择 Delete

  4. 在弹出的确认框中选择 Yes 即可删除对应 skin

查找 Skins 文件的本机位置

  1. 点击标准工具栏左侧的 Output 选项卡

  2. 选择左侧的 skins

  3. 选中需要删除的 skin，点击  并选择 Show in Explorer 即可查找 Skin 文件的本机位置

使用 Lables

什么是 Lables

Lables 是帮助系统（输出）的各个元素的显示文本。例如，browse sequence 中上一个和下一个按钮、显示/隐藏以及其他按钮、选项卡、工具提示和一些消息的文本内容。在 PDF 输出中，Lables 是 TOC、Index 和 Glossary 页面的标题。

项目内容外的输出文本内容来自 Lable 文件。Lable 文件是基于特定语言的，并且包含按不同类别分组的 lables，例如 common，browse sequence 和 master pages。每个类别都列出了适当的 lables 及其值。

RoboHelp 使用默认语言（与项目语言相同）添加 Lable 文件。通过添加、导入或导出并编辑 Lable 文件，可以自定义 lables 语言。

添加 Lables

  1. 点击标准工具栏左侧的 Output 选项卡

  2. 选择左侧的 lables

  3. 选择 lables 右上角标记 

  4. 在弹出的选项框中选择语言即可添加对应的 Lables 文件

编辑 Lables

  1. 点击标准工具栏左侧的 Output 选项卡

  2. 选择左侧的 lables

  3. 选中需要编辑的 lable，点击  并选择 Edit

删除 Lables

  1. 点击标准工具栏左侧的 Output 选项卡

  2. 选择左侧的 lables

  3. 选中需要删除的 lable，点击  并选择 Delete

  4. 在弹出的确认框中选择 Yes 即可删除对应 lable

导入 Lables

  1. 点击标准工具栏左侧的 Output 选项卡

  2. 选择左侧的 lables

  3. 选中一个 lable，点击  并选择 Import 即可导入 lable 文件（.lng 格式）

导出 Lables

  1. 点击标准工具栏左侧的 Output 选项卡

  2. 选择左侧的 lables

  3. 选中需要导出的 lable，点击  并选择 Export 即可导出 lable 文件到指定的文件夹

使用 Master Pages

什么是 Master Pages

HTML 页面包含三个元素：内容，布局和样式。尽管 CSS 文件将样式与内容分开，但布局仍嵌入 HTML 代码中。 通过使用 Master Pages，可以将布局与样式和内容分开。

Master Pages 包含布局信息，并与CSS文件相关联。 它是HTML主题的模板，为页眉、页脚和 Topic 定义占位符，并且可以使用 breadcrumbs、Mini TOC、snippets、variables、fields、symbols 和 condition tags 。

你可以使用 Master Pages 创建 Toic，也可以将现有 Topic 与 Master Pages 关联。 使用 Master Pages 创建 Topic 时， Topic 的占位符内容将放置在最终 Topic 中。 如果将 Master Pages 应用于现有 Topic ，则将忽略 Topic 占位符内容。 在这种情况下， Topic 的实际内容会在预览和生成时替换 Master Pages 中的 Topic 占位符。

添加 Master Pages

  1. 点击标准工具栏左侧的 Author 选项卡

  2. 选择左侧的 Mater Pages 

  3. 选择 master pages 右上角标记 

  4. 在弹出的选项框中填写相关信息，点击 create 即可添加 master page

编辑 Master Pages

编辑 master page 时，可以使用 Breadcrumbs、Variables、Snippets、Mini TOC、Fields、Symbols 和 Condition Tags。

  1. 点击标准工具栏左侧的 Author 选项卡

  2. 选择左侧的 Mater Pages 

  3. 选中需要编辑的 master page，点击  并选择 Edit

使用 Breadcrumbs

  • 点击标准工具栏的 Insert Breadcrumbs Placeholder  

  • 选中 master page 中的 breadcrum 进行进一步编辑

使用 Mini TOC

  • 点击标准工具栏的 Insert Mini TOC Placeholder   

  • 选中 master page 中的 Mini TOC 进行进一步编辑

使用 Field

  • 点击标准工具栏的  Insert Variables/Snippets/Fields ，并选择 Field

  • 弹出的对话框提供插入日期、时间、文件等 Field 

使用 Symbol

  • 点击标准工具栏的  Insert Special Character  

  • 弹出的对话框提供插入标点、货币、扩展 ASCII 等标记 

删除 Master Pages

  1. 点击标准工具栏左侧的 Author 选项卡

  2. 选择左侧的 Mater Pages 

  3. 选中需要编辑的 master page，点击  并选择 Delete

  4. 在弹出的确认框中选择 Yes 即可删除对应 master page

查找 Master Pages 文件的本机位置

  1. 点击标准工具栏左侧的 Author 选项卡

  2. 选择左侧的 Mater Pages 

  3. 选中需要编辑的 master page，点击  并选择 Show in Explorer 即可查找 Master Pages 文件的本机位置
