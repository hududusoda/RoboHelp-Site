创建和管理内容

 

Topic 是具体文档内容（文本、图像、多媒体、链接等）的容器，通过增加、编辑、排版、删除 topic 的内容实现对整个文档的构建。可以创建不同的文件夹来管理多个 topic。

创建文件夹

可以在 Contents 面板的根目录下创建文件夹，也可以在文件夹创建子文件夹。 

在打开的项目中创建根目录文件夹：

  1. 在 Author 工具栏中单击 Contents

  2. 在 Contents 面板中，单击面板右上角的，然后单击 New Folder

  3. 在 Folder Details 对话框中，在“ 名称”中指定文件夹  名称。

  4. 点击 Create

创建 Topic

可以在文件夹内或 Contents 面板的根级别上创建多个 topic，也可以在文件夹内创建多个 topic。

在文件夹中创建主题：

  1. 在 Author 工具栏单击 Contents

  2. 在 Contents 面板中，执行下列操作之一：

    1. 选择要存放 topic 的文件夹，单击面板右上角的 ，单击 New Topic

    2. 右键单击要存放 topic 的文件夹，选择 New > New Topic

    注意：如果没有选择文件夹，则默认在根级别创建 topic。

  3. 在 New Topic 对话框中，根据需要设置 topic 属性

  4. 点击 Create

编辑 Topic 内容

  1. 在 Author 工具栏中单击 Contents

  2. 在 Contents 面板中，展开包含要编辑的 toppic 文件夹，双击该 topic 打开

  3. 在 topic 中可以添加或编辑文本、图像和多媒体内容，可以创建跳转至 topic 内部某处或外部内容的连接，可插入或自定义 Mini TOC、variables、snippets、 fields 等

    注意：要快速搜索要编辑的 topic，单击左上方工具栏的  或键入 Ctrl + F，在搜索对话框中输入 topic 标题和文件名，在搜索结果中双击 topic 打开。

  4. 要在打开的 topic 中启用拼写检查，请执行以下操作：

    1. 选择  Edit > Preferences  或键入 Ctrl + Shift + T

    2. 在 Application Settings 对话框中，选择 Enable Spell Checking

    3. 单击 Done

  5. 单击 关闭当前 topic

    注意：关闭之前请务必保存修改！

导入文件和文件夹

  1. 在 Author 工具栏中单击 Contents

  2. 导入文件：

    1. 右键单击一个文件夹或单击该文件夹的 ，选择 Import> File

    2. 在 open 对话框中，选择要导入的文件，然后单击 Open

  3. 导入文件夹：

    1. 右键单击一个文件夹或单击该文件夹的 ，选择 Import> Folder

    2. 在 open 对话框中，选择要导入的文件夹，然后单击 Select Folder，所选文件夹及下面的所有文件将复制到该项目文件夹中 

链接文件和和文件夹

  1. 在 Author 工具栏中单击 Contents

  2. 链接文件：

    1. 右键单击一个文件夹或单击该文件夹的 ，选择 Link > Link File

    2. 在 open 对话框中，选择要导入的文件，然后单击 Open，该文件将复制到所选文件夹中，RoboHelp 会提示副本文件/项目文件已链接到源文件，Content 面板中项目文件前的显示绿色箭头，表示同步状态

      注意：如果在源文件中进行了修改，则箭头由绿色变为红色，表明状态不同步。
      要同步链接的文件，执行以下操作：
      •    从源到项目：右键单击 topic 或单击该 topic 的 ，然后单击 Sync > From Source to Project
      •    从项目到源：右键单击 topic 或单击该 topic 的 ，然后单击 Sync > From Project to Source 
      之后会出现同步成功或失败的通知。
      要删除项目文件与源文件之间的链接，右键单击 topic 或单击该 topic 的 ，然后单击Remove Link，用于表示同步状态的箭头也会随之消失

  3. 链接文件夹：

    1. 右键单击一个文件夹或单击该文件夹的 ，选择 Link> Folder

    2. 在 Open 对话框中，选择要导入的文件夹，然后单击 Select Folder，所选文件夹及下面的所有文件将复制到该项目文件夹中 

创建 HTML Topic

注意：仅适用于 Word 和 FrameMaker 文件。

  1. 在 Author 工具栏中单击 Contents

  2. 要从 Word 或 FrameMaker 文件创建HTML主题，右键单击一个文件夹或单击该文件夹的  ，然后单击 Update HTML， 
    RoboHelp 将创建一个与 Word 或 FrameMaker 文件同名的文件夹，用于存储所有HTML topic和引用的资源

    注意：创建HTML topic 时，RoboHelp 会遵循 Project Settings 对话框（General 选项卡）中以下字段指定的设置：
    •    Word：Default Word Import Settings
    •    FrameMaker：Default FrameMaker Import Settings
    如果一个文件夹包含多个文件，可以通过以下从所有文件同时创建HTML topic：
    •    Word文件：右键单击该文件夹，然后选择 Word Documents > Update All.
    •    FrameMaker文件：右键单击该文件夹，然后选择 FrameMaker Documents> Update All

重命名 Topic 和文件夹

  1. 在 Author 工具栏中单击 Contents

  2. 若要重命名文件夹，则右键单击这个文件夹或单击该文件夹的 ，然后单击 Rename；若要重命名 topic，则先要展开该 topic 所在的文件夹，右键单击该 topic 或单击该 topic 的 ，然后单击 Rename

  3. 在 Rename 对话框中，输入新标题和新文件名，然后单击 Done

创建 Topic 副本

  1. 在 Author 工具栏中单击 Contents

  2. 要创建 topic 副本，则先要展开该 topic 所在的文件夹，右键单击该主题或单击该主题，然后单击 Duplicate

  3. 在 Duplicate 对话框中，为副本设置标题和文件名，然后单击 Done，Robohelp 就会在同一文件下创建副本

 删除 Topic 和文件夹

  1. 在 Author 工具栏中单击 Contents

  2. 若要删除文件夹，则右键单击这个文件夹或单击该文件夹的 ，然后单击 Delete；若要删除 topic，则先要展开该 topic 所在的文件夹，右键单击该 topic 或单击该 topic 的 ，然后单击 Delete

  3. 在 Confirm 对话框中单击 Yes，该 topic 或文件夹被删除
