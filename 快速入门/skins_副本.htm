编辑格式

 

在 Adobe RoboHelp 中创建 topic后，即可在其中添加文本、图片、表格、视频、链接等，并设置这些内容的格式，还能添加符号、HTML 元素、扩展文本、下拉文本等

使用标准工具栏中的格式设置选项和 Properties 面板中的 General 选项卡来进行排版和布局。单击屏幕右上角的 Preview 快速预览呈现效果。

应用格式

以下两处可进行格式设置：

  • 标准工具栏

  • Properties 面板中的 General 选项卡

标准工具栏

标准工具栏用于向 topic 添加不同形式的内容，包含以下设置选项：

图标 格式化选项 描述

项目符号列表样式

将内容转为无序列表

编号列表样式

将步骤描述、活动过程和任务描述转为有序列表

多级列表样式

将内容格式化转为多个级别的无序或有序列表

插入表格

添加表格，可以在表格中整理文字和多媒体，并使用不同的方式突出显示它们颜色，并执行更多操作

插入图片

添加各种格式的图片，如 PNG、JPEG、WEBP 和 SVG 等，支持从项目、本地文件夹、网页添加图片

插入多媒体

从本机、YouTube、Vimeo、Adobe Captive导入视频，包括 Adobe Captivate 视频

插入链接

向文中添加超链接，连接到其他 topic 内容或外部内容（如本地文件、电子邮件和网页等）

插入书签

选择内容并将其作为书签插入，书签也可以用来创建 topic 内链接

插入 Variables/Snippet/Fields

将 topic 中的 variables、snippet、fields 添加到同源内容，并快速更新不同类型的输出

插入HTML元素

添加HTML元素，如 Horizontal Line、iframe 和 Text Box，来控制内容的外观和布局。

插入符号

在文本中添加符号，包括数学符号、图形、图标

创建扩展文本

在特定内容部分附近添加额外的文本，例如在最终文档中点击高亮文字，会有其他内容显示

创建下拉文本

在特定内容部分添加下拉文本，例如在最终文档中点击高亮文本，会有下拉文本出现，再次点击隐藏

插入 Mini TOC 占位符

在topic或 masterpage中插入 Mini TOC 占位符

 

Properties面板中的 General选项卡

 

在此设置字体、字号、字符间距、缩进、颜色等，还可以添加边框、自定义背景。

  1. 在 Author 工具栏中单击 Contents

  2. 在 Contents面板中，双击打开一个 topic，在屏幕右侧查看 Properties面板

  3. 在 topic中，单击选中一个元素，在 Type 字段中查看其类型。例如，选中 topic 标题，显示为 h1（Heading 1）,将一个段落显示为 p
    ‎对于在 Type 字段中显示的选定元素：

    • 可以编辑格式，然后应用于整个项目中性质相同的所有元素，并更新样式表：单击 Type 字段旁的 Update Style 图标。 例如，根据样式表，topic 标题的默认字体为 Arial 20 磅，将字体更改为 Adobe Clean 24 磅后，要应用于项目中的所有 topic 标题，并更新样式表，单击即可

    • 可以清除格式：选择文本，单击 Type 字段旁的图标 或选择 Edit > Clear Formatting 来清除当前格式。例如，根据样式表，topic 标题的默认字体为 Arial 20 磅，你已将字体更改为 Adobe Clean 24 磅，但未更新样式表，要将默认字体恢复为样式表默认的 Arial 20 点，选择文本后单击  即可

  4. 要设置文本格式，单击 Font下拉列表，该列表包括以下选项

    • Font Family     选择字体

    • Font Style     设置字体样式，例如斜体、加粗等。

    • Font Size     手动输入字体大小，输入所需的字体大小，或使用递增和递减按钮以0.5单位更改当前大小，在单位的下拉列表中选择单位，例如像素，百分比等

    • Color     单击  设置字体颜色，也可在此字段中输入16进制RGB编号

    • Text Transform     大小写转换

    • Inline formatting options     设置内联格式，如删除线、下划线、加粗、斜体、上标和下标。

    • Character Spacing     设置字符间距，或使用递增和递减按钮以0.5单位更改当前字符间距，在单位的下拉列表中选择单位，例如像素，百分比等

  5. 要更改内容的对齐方式，请单击 Alignment 下拉列表，该列表包括以下选项：

    • Alignment Options     左、居中、右对齐

    • Line Height     在下拉列表中设置行高>

    • Decrease/Increase Indent     通过增加或减小的按钮更改内容和边缘的间距

    • Border Style     设置边框样式，如实线、虚线、双实线等

    • Color     单击  设置边框颜色，也可在此字段中输入 16 进制 RGB 编号

    • Width    设置边框宽度，然后在 Unit下拉列表中选择单位

  6. 要设置内容的边框，单击 Border 下拉列表，该列表包括以下选项：

    • 以下图标表示将边框应用于特定的内容边缘：
        应用于所有边缘
        仅应用于顶部边缘
        仅应用于右边缘
        仅应用于底部边缘
        仅应用于左边缘

  7. 要对页面内容进行布局，单击 Layout下拉列表，该列表包括以下选项：

    • Height     设置内容的高度，可以用 Increment 和 Decrement 按钮来增加或缩短高度，然后在 Unit 下拉列表中选择单位

    • Width     设置内容的高度，可以用 Increment 和 Decrement 按钮来增加或缩短宽度，然后在 Unit 下拉列表中选择单位

    • Margin （外边距）

      • 在 Unit 下拉列表中选择 margin 单位

      • 单击   更新四个 margin 值，但不会同步应用于项目所有 margin 值

      • 单击  将更新的四个 margin 值同步应用于项目所有 margin 值

      • 设置顶部、底部、左侧和右侧的 margin 值，使用 Increment 和 Decrement 按钮来增加或减少四个 margin 值

    • Padding （内边距）

      • 在 Unit 下拉列表中选择 padding 单位

      • 单击  更新四个 padding 值，但不会同步应用于项目所有 padding 值

      • 单击  将更新的四个 padding 值同步应用于项目所有 padding 值

      • 设置顶部、底部、左侧和右侧的 padding 值，使用 Increment 和 Decrement 按钮来增加或减少四个padding值

    • Float     用于元素在网页上定位和布局。在下拉列表中选择 float 值：

      • left 和 right     表示容器中内容的浮动方向

      • none     表示该元素不浮动，显示在文本中

      • inherit     表示该元素继承其父元素的float值

    • Clear     用于指定可以在清除的元素旁浮动的元素，以及在哪一侧浮动。在下拉列表中选择 clear 值：

      • inherit     表示该元素继承其父元素的 clear 值

      • left 和 right     表示左侧或右侧不可使用浮动元素

      • none     表示左侧和右侧均可使用浮动元素

      • both     表示左侧或右侧均不可使用浮动元素

  8. 要设置内容背景，单击 Background 下拉列表，该列表包括以下选项：

    • Shading     单击  设置背景颜色，也可在此字段中输入 16 进制 RGB 编号

    •       插入本机图片作为背景图像

    • Width     设置背景图像的宽度

    • Height     设置背景图像的高度

    •      保持纵横比

  9. 要复制所选内容的格式，先选中要应用复制格式的内容，然后单击 Edit > Copy Formatting

  10. 要删除所选内容的格式，先选中要删除的内容，单击 Edit > Clear Formatting，所选内容会以默认格式显示

  11. 要保存更改，单击标准工具栏中的全部保存图标 

使用列表

插入列表

  1. 选择要转为列表形式的文本，然后执行以下操作之一：

    • 创建项目符号列表，单击标准工具栏中的 ，然后在下拉列表中选择符号样式

    • 创建编号列表，单击标准工具栏中的 ，然后在下拉列表中选择编号样式

    • 创建多级编号列表或多级项目符号列表，请执行以下操作：

      • 单击标准工具栏中的 ，然后在下拉列表中选择样式

      • 要创建子列表，先选中文本，然后按 Tab 键

  2. 要删除已有的列表格式，先选中列表内容，单击标准工具栏中的 、 或 

使用表格

插入表格

  1. 将光标置于需要插入表格的位置，然后单击标准工具栏中的 

  2. 在下方弹出的网格中拖动鼠标，设置行列数，选定的行列会蓝色高亮显示，单击确定选中，在 topic 中插入表格

  3. 右键单击表格，在选项中可以进行以下操作：

    • Cell     插入、删除、合并和拆分单元格。

    • Row     插入或删除行

    • Column     插入或删除列

    • Delete Table     删除表格

    • Select Table     选中有内容的单元格，不选中空单元格

    • Table Properties      在 Properties 面板 > General 选项卡中查看表属性，可以设置表格和行列的宽度和高度，也可以在表格上方或下方添加标题

    • Apply Condition Tags     将 condition tag 应用到表格。在 Apply Condition Tags 对话框的左窗格中，选择要应用的 condition tag，单击 Apply

使用扩展文本

插入扩展文本

  1. 在 topic 中选中要扩展的文本，然后单击标准工具栏中的，所选文本成为扩展文本的标题，其后出现扩展文本的占位符

  2. 在占位符中输入扩展文本内容

  3. 通过以下方式管理扩展文本：

    • 切换 topic 中所有扩展文本的占位符边界显示：选择 View > Show > Expanding/Drop-down Text Boundary

    • 切换特定扩展文本占位符的显示：右键单击文本标题，选择 Show Expanding Text or Hide Expanding Text 

    • 删除扩展文本：右键单击文本标题，选择 Remove Expanding Text 

    • 折叠或展开扩展文本：双击扩展文本标题

    • 折叠或展开 topic 中所有扩展文本：右键单击任何一个已展开的扩展文本标题，然后选择 Collapse All Text 或 Expand All Text

    • 为处于折叠状态的扩展文本标题设置单独的文本：单击扩展文本，在 Properties 面板 > General 选项卡 > Text when collapsed 字段中，输入新标题。在最后生成的文档中，默认情况下扩展文本是处于折叠状态的，此处输入的新标题将在扩展文本隐藏的状态下显示，扩展文本展开时会覆盖此处内容

  4. 单击标准工具栏左侧的  保存全部修改

使用下拉文本

插入下拉文本

  1. 在 topic 中，选择要下拉的文本，然后单击标准工具栏中的单击 ，所选文本成为下拉文本的标题，下方出现下拉文本的占位符

  2. 在占位符中输入下拉文本的内容

  3. 通过以下方式管理下拉文本：

    • 切换 topic 中所有的下拉文本占位符的边界显示：选择 View > Show > Expanding/Drop-down Text Boundary

    • 切换特定下拉文本占位符的显示：右键单击文本标题，选择 Show Drop-down Text 或 Hide Drop-down Text

    • 删除下拉文本：右键单击文本标题，选择 Remove Drop-down Text

    • 折叠或展开下拉文本：双击文本标题

    • 折叠或展开 topic 中的所有下拉文本：右键单击任何一个已展开的扩展文本标题，然后选择 Collapse All Text 或 Expand All Text

    • 为处于折叠状态的下拉文本标题设置单独的文本：单击下拉文本，在 Properties 面板 > General 选项卡 > Text when collapsed 字段中，输入新标题。在最后生成的文档中，默认情况下下拉文本是处于折叠状态的，此处输入的新标题将在下拉文本隐藏的状态下显示，扩展文本展开时会覆盖此处内容

  4. 单击标准工具栏左侧的保存全部修改

添加和编辑 Twisty Images

Robohelp 支持向扩展和下拉文本以及 topic TOC 添加图片，即 Twisty Images，图片分别用于展开和折叠状态。在预览和最后生成的文档中，相应图片会以展开和折叠状态显示。

添加Twisty Image

  1. 在 topic 中单击一个扩展或下拉文本，或在 topic TOC 中单击 Caption 一词

  2. 在 Properties 面板 > Styles 选项卡中，在下拉列表中选择 Hyperlink Styles

  3. 在显示的超链接样式列表中，选择一种样式应用于所选项目，只要应用的样式设置了 twisty 功能，twisty image 就会在预览和最后生成的文档中显示

  4. 请执行以下任一操作：

    • 在应用的样式上单击  进行编辑
      ‎在样式表编辑器的Styles面板中的Hyperlink Styles里，应用的样式开放编辑

    • 在 Contents 面板中，展开 assets > css，双击一个样式表，或单击该样式表的 ，展开 Hyperlink Styles，然后选择应用的样式

  5. 在样式表编辑器的 Properties 面板中，展开 Twisties

  6. 在 Image when expanded 和 Image when collapsed fields 字段，单击  以选择并插入所需的图片

  7. 单击  将图片置于文本的开头（默认）或  将图像置于文本的末尾 

  8. 单击标准工具栏左侧的  保存全部修改

注意：在为特定的超链接样式设置 twisty image 之后，只需在 Styles 面板中应用相同的超链接样式，就可以将相同的twisty image 应用于不同的项目
