输出文件

 

本章介绍RoboHelp中不同类型的输出预设，以及如何使用它们生成各种格式的输出。例如响应 HTML5、WebHelp、Mobile App 等。

RoboHelp具有强大的发布功能，因此您可以生成多种格式的输出。从各种输出预设类型中进行选择，例如响应HTML5、PDF 和 Mobile App，并满足来自各种平台的用户的需求。输出预设方便易用，可以使用多种设置从同一项目创建具有不同内容、布局和用户体验的多个帮助系统。

输出预设的类型

可以使用所需的输出预设类型来生成项目输出。 RoboHelp 中提供以下类型的输出预设：

  1. 自适应 HTML5

  2. PDF 格式

  3. Microsoft HTML Help Web帮助

  4. 移动应用

  5. 电子书

使用输出预设来生成输出

在 RoboHelp 中生成项目输出之前，需要创建和配置输出预设。准备好输出预设后，即可使用该预设来生成输出。

创建输出预设

要在RoboHelp中创建输出预设，请按照下列步骤操作：

  1. 在一个打开的项目中，执行下列操作之一：

    • 单击 Standard toolbar 工具栏右上角的 Quick Generate，然后单击 Edit Settings 图标

    • 单击 Standard toolbar 工具栏左侧的 Output 选项卡，输出视图将会打开

  2. 在输出工具栏中，单击 Output Presets，将打开 Output Presets 面板

  3. 在 Output Presets 面板中，单击 

  4. 在 New Preset 对话框中，指定以下内容：
    Name 指定输出预设的名称
    Type 使用下拉列表选择所需的输出预设类型。你可以从自适应 HTML5、PDF、Microsoft HTML Help、WebHelp、移动应用程序和电子书中选择

  5. 单击完成

可以从 Output 工具栏中的 Output Presets 面板访问输出预设。也可以在创作窗口的 Quick Generate 对话框中访问输出预设。

配置输出预设

  1. 在一个打开的项目中，执行下列操作之一：

    • 单击 Standard toolbar 右上角的 Quick Generate，然后单击 Edit Settings 图标

    • 单击 Standard toolbar.左侧的 Output 选项卡

  2. 在输出工具栏中，单击输出预设。在 Output Presets 面板中，执行下列操作之一：

    • 双击所需的输出预设

    • 单击所需的输出预设旁边的图标，然后选择编辑

  3. 根据您选择的输出预设的类型，您可以查看 General、Content 等标签，以配置设置

  4. 要保存设置，请单击标准工具栏左上角的全部保存图标

产生输出

要生成输出，请按照下列步骤操作：

  1. 在打开的项目的创作窗口中，单击标准工具栏右上角的 Quick Generate

  2. 在 Quick Generate 对话框中，选择所需的输出预设，然后单击 Generate

进度条显示输出生成的进度。要取消生成输出，请单击。输出生成完成后，要查看输出，请单击。如果输出生成失败，请单击以查看错误日志。

注意：
在生成输出之前，您还可以通过 Quick Generate 对话框编辑多个输出预设。为此：
选择要编辑的输出预设，然后单击 Edit。每个输出预设的预设编辑器会打开。

使用命令行生成输出

要在不启动 RoboHelp GUI 的情况下生成输出，请在命令行实用程序中指定以下命令：

<path_of_Robohelp.exe> --cl "<project_path>" -o "<output_preset>" -l "<logfile_path>" -p "<output_path>"

命令说明：

  • path_of_Robohelp.exe RoboHelp 可执行文件的路径

  • --cl 在命令行模式下启动 RoboHelp

  • project_path 要生成的项目的路径

  • -o 包括输出预设

  • output_preset 要用于生成输出的输出预设的名称

  • -l或--log（可选） 包括日志文件参数

  • logfile_path 如果包括 log file 参数，请指定在其中生成日志的日志文件的路径。日志文件必须采用 HTML 格式，因为日志采用 HTML 格式

  • -p 或 --path（可选） 包括输出路径参数

  • output_path 如果包含输出路径参数，请指定要保存生成的输出的路径。如果您排除此参数，则 RoboHelp 会在所选输出预设的属性中指定的输出路径上生成输出

管理输出预设

可以轻松地复制可用的输出预设，也可以从Output Presets面板和Quick Generate 对话框中将其删除。

重命名输出预设

  1. 单击 Standard toolbar 左侧的 Output 选项卡。将打开 Output 视图

  2. 在Output toolbar,中，单击Output Presets。将打开 Output Presets面板

  3. 在 Output Presets 面板中，针对要重命名的输出预设，选择 Options > Rename

  4. 在 Rename Preset 对话框中，指定 Output Presets 的名称

  5. 单击完成

复制输出预设

  1. 请执行以下任一操作：

    • 单击标准工具栏右上角的 Quick Generate，然后单击 Edit Settings 图标

    • 单击 Standard toolbar 左侧的 Output 选项卡
      Output view 将打开

  2. 在输出工具栏中，单击输出预设。将打开“输出预设”面板

  3. 在“输出预设”面板中，单击所需的输出预设旁边的“选项”

  4. 在“选项”下拉列表中，选择“重复”

  5. 指定重复的输出预设的名称，然后单击完成

删除输出预设

  1. 请执行以下任一操作：

    • 单击标准工具栏右上角的 Quick Generate，然后单击 Edit Settings 图标

    • 单击 Output 工具栏左侧的 Standard toolbar 选项卡。将打开 Output 视图

  2. 在 Output toolbar 中，单击输出预设。将打开 Output Presets 面板

  3. 在 Output Presets 面板中，选择所需的输出预设，然后单击 Options 

  4. 在Options 下拉列表中，选择 Delete。在 Confirm 对话框中，单击 Yes

在计算机中找到输出预设文件

  1. 单击 Standard toolbar 左侧的 Output 选项卡。将打开 Output 视图

  2. 在 Output toolbar 中，单击 Output Presets。将打开 Output Presets 面板

  3. 在 Output Presets 面板中，针对要查找的输出预设，选择 Options > Show in Explorer

Windows 资源管理器将打开，以显示输出的预设文件及其在计算机中的位置
