软件界面

 

Adobe RoboHelp 具有许多强大的功能，提供经过精心设计、简介且易用的界面。主要特色有：

  1. 快速创建具有所需所有元素的内容

  2. 通过同源开发和内容复用来提高效率

  3. 以所需的输出类型发布项目

  4. 同一项目中可设置多个合作者

RoboHelp 2019 界面

A. 顶部菜单栏    B. Author 工具栏    C. 项目组件面板    D. Open topic     E. 属性面板     F.  标准工具栏

顶部菜单栏

顶部菜单栏提供编辑、查看文件、协作和访问帮助菜单功能。File 可以创建、打开、保存和关闭项目，支持导入 Word 文档和 Captivate 视频、升级 RoboHelp Classic 项目、配置项目设置以及退出项目；Edit 支持剪切、复制、粘贴、撤消和重做内容，提供全选、复制、清除格式以及编辑应用程序功能；通过 View 可查看 Properties 面板、condition tags、variables、按名查看 topic；Collaborate 可以将项目设置为公开协作模式；Help 有助于访问帮助资源、管理帐户。

Author 工具栏

Author 工具栏位于屏幕的左侧，在标准工具栏下方。 通过访问项目的各个组件，可轻松管理项目。

项目组件面板

单击 Author 工具栏上的项目组件时，将打开项目组件面板，该面板可管理该组件。可以在 Author 工具栏中找到以下项目组件：

  • Contents 面板列出了项目中的主题，并提供了添加、重命名和编辑主题属性的选项，也可以通过创建文件夹和子文件夹来组织主题

  • Table of Contents 面板包括项目中文件夹，子文件夹和主题的层次结构；支持重新排序文件夹并以战略性方式组织内容，以便在输出中轻松进行用户导航；支持编辑、重命名和删除目录

  • Index 面板列出项目中的索引，支持添加、重命名、编辑和删除索引

  • Glossary 面板列出了项目中的词汇表术语，支持添加、编辑、重命名或删除词汇表文件

  • Condition Tags 面板列出了项目中分配的条件标签，支持添加、编辑和删除 condition tags

  • Variables 面板显示在项目中分配了值的所有变量，支持添加、编辑或删除 variable

  • Snippets 面板显示项目中的代码片段，支持创建、编辑或删除片段

  • Master Pages 面板显示项目中的 master page，支持创建、编辑、复制、删除、导入或导出 master page

  • Browse Sequences 面板显示项目的浏览序列，支持创建，编辑，重命名或删除定义浏览序列的文件

  • Reports 面板包括生成报告的选项、例如 topic 列表、项目状态等，支持编辑、下载和打印所需的报告

当前 Topic

在 Contents 面板中双击一个主题以打开一个主题，可以编辑 topic 中的内容并设置其格式。 

Properties 面板

在项目中创作主题时，屏幕右侧显示的面板是 Properties 面板。 该面板包含以下选项卡：

General：设置字体、对齐方式、边框、布局和背景等常规样式

Styles：编辑 topic 的元素属性

Topic：调整所选对象的属性时，可以实时查看内容的变化；通过选择 View > Properties 面板来隐藏或显示 Properties 面板，或者单击标准工具栏上的窗格切换图标

Standard 工具栏

Standard 工具栏出现在菜单栏下方，并包括三个工具集：

  1. 标准工具栏左侧的一系列选项卡和图标，包括 Author 和 Output 选项卡，以及保存所有、撤消、重做、查找文件以及查找和替换图标

  2. 标准工具栏中的图标集，包含各种快速排版布局内容的选项，如插入项目符号和编号列表、图像、多媒体等

  3. 标准工具栏右侧的图标集，包含各种视图选项，例如 Author、Source 和 Preview，还包括 Quick Generate 和 Toggle Right Panel；Quick Generate 对话框允许选择和配置输出预设并生成输出
