RoboHelp 2019 介绍

 

注意：本产品不支持 Mac OS 操作系统。

Adobe RoboHelp（2019）是一款用于创建和发布内容的工具，它功能强大且操作直观，可以帮助您轻松高效地开发帮助系统。通过直观易用的功能你可以用最少的精力为不同的受众和各种设备提供定制的内容。RoboHelp 包含强大的内容编辑、布局创建、格式设置和搜索功能，可帮助你快速创建并快捷高效的管理内容。

RoboHelp 提供现代化的响应式 HTML5 和设备友好型的输出预设，来创建个性化的内容。针对不同的受众，可以使用 Condition Tags 和 Condition expression 创建交互式和定制化内容，便于迎合多样化的受众群体。

RoboHelp 2019 入门

主要特性

响应式 HTML5 和 CSS3 创作支持

借助现成的 HTML5 和 CSS3 创作支持，使用文本和多媒体创造高度沉浸式内容体验；轻松添加来自 YouTube 和 Vimeo 视频、音频、图像、文字效果、符号和其他元素，使内容引人入胜；强大的预览功能，便于在创作时实时观察效果。

借助 SVG 丰富内容

使用 SVG，有效保证内容在移动设备、Web 和打印设备上的高清晰度。支持在SVG 文件中导入并使用多个视觉效果，从而保证最终输出结果拥有最佳用户体验。

智能内容重用

使用 Variables 和 Snippets 在项目中进行跨 topic 的内容复用，节省时间和精力；Variables 和 Snippets 的任何更新都会同步至整个项目。

有条件的内容定义和发布

使用 Condition Tags 创建规则，并应用于 topic、段落，甚至词句，从而实现同一项目生成不同的内容。

整洁的项目结构

RoboHelp 以整洁有序的结构维护所有项目内容。无论是在 RoboHelp 工作区中还是在计算机资源管理器视图中，都可以在单独的文件夹中管理项目资产（如样式表和代码片段）。

多视图切换

使用屏幕右上角的图标时，可以在作者 视图、代码视图和预览视图之间轻松切换，便于快速查看内容输出及其 HTML 效果。

直观的对象属性设置界面

对象属性的设置界面非常直观，在创作内容时，不仅可以简便的调整添的各种对象属性而且可以实时预览。选择添加的对象后，右侧的 Properties 面板将显示所选对象的所有属性。

强大的 Skin 编辑器

借助功能强大的 RoboHelp 皮肤编辑器，可以创建更具吸引力和用户友好性的现代化界面。

优化的搜索引擎

内置的智能算法能为搜索引擎提供一流的搜索结果，并高亮显示搜索到的单词。

多格式发布

使用可立即将内容发布为各种输出格式——响应式 HTML5、Microsoft HTML Help (CHM)、WebHelp 和 Mobile App 等。可以一键获取持有 PhoneGap Build 许可证的 iOS 和 Android 应用程序。

下一代自适应 HTML5 布局

通过新的响应式 HTML5 布局提供个性化的内容体验、直观的导航栏以及一流的搜索功能。通过这样的布局可以轻松显示或隐藏词汇表和目录等小部件。

更高的内容可访问性

使用 508-compliant HTML5 布局来创建可发布在多个平台（例如 Web、移动端和印刷品）的内容，以满足最终用户的特殊需求。

优化的 HTML5 输出

与此前的 RoboHelp 版本相比，现版本的加载时间减少了 50％，每个 topic 可节省约 20 KB 的内存空间。基于优化的 HTML5，响应速度更快，页面之间过渡平滑。

添加缩略图

图像可发布为缩略图，并在单击时扩展全图，迎合了目标用户在移动设备上查看内容的需求。

一键生成移动应用程序

一键生成通过 PhoneGap Build 认证的移动应用程序。

Git 支持

支持 Git 版本控制和协作，团队成员能够快速高效地在各种规模的项目上进行协作。

SharePoint Online 支持

使用 SharePoint Online 在整个组织中实现共享和协作，确保每个人都可以编辑同一页面。

 

基本工作流程

RoboHelp 中的基本工作流程：从创建项目到输出文档成品

创建项目

一个项目就像一个容器，用于存放所有内容和各种组件，例如目录、摘要、样式等。

编辑内容

在 RoboHelp 强大的编辑器中，创建 topic 并添加文本，图像和多媒体，使用文本效果、表格、列表等对内容进行排版。

导入内容

将其他来源的内容（如 Microsoft Word文件）导入项目。

建立导航

将文本和图像链接到其他内容，并在项目中确定内容层次结构或组织方案。

配置输出设置

在 RoboHelp 中选择输出预设的类型（如响应式 HTML5 或 PDF），并配置输出的外观和功能；还可以根据最终用户的兴趣、技能水平和其他因素来设置内容的显示或隐藏状态。

输出文档成品

生成或发布多个平台（例如Web，移动端和印刷品）的输出。可以从各种输出预设中进行选择，如响应式 HTML5、PDF、Microsoft HTML Help (CHM)、WebHelp、移动应用程序等等。
